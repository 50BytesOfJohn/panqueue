# Panqueue

Panqueue is a modern queue library, currently designed specifically for Deno

# Development

Please note that library is not released yet. This means that we do not need backward compatibility. Trying to do not break current logic can lead to over complicated solutions, so remember that this is not needed. Make breaking changes, change core stuff if needed.

* We use clean and readable code
* We focus on performance, queue library needs to be fast and light
* We want to follow modern best practices and language APIs to get most of the runtime
* DX is crucial part of Panqueue
* We add comments to make logic and decisions clear for contributors, as well as good JS docs for developers integrating Panqueue
* We do not over engineer. Simplicity and balance. Sometimes perfect code is not necessery.
