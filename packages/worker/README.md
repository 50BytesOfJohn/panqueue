# @panqueue/worker

Worker pool and scheduler for processing [Panqueue](https://github.com/50BytesOfJohn/panqueue) jobs from Redis — lease-based stalled-job recovery, atomic state transitions, force-shutdown with atomic requeue.

## Install

```sh
npm install @panqueue/worker   # or pnpm / yarn / bun
deno add jsr:@panqueue/worker  # Deno (JSR)
```

## Documentation

See the [Panqueue repository](https://github.com/50BytesOfJohn/panqueue) for full documentation.
