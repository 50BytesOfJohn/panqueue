# @panqueue/core

Shared types, key helpers, and serialization primitives for [Panqueue](https://github.com/50BytesOfJohn/panqueue) — runtime-agnostic Redis job queues for JavaScript and TypeScript.

You usually don't depend on this package directly; it backs [`@panqueue/client`](https://www.npmjs.com/package/@panqueue/client) and [`@panqueue/worker`](https://www.npmjs.com/package/@panqueue/worker).

## Install

```sh
npm install @panqueue/core   # or pnpm / yarn / bun
deno add jsr:@panqueue/core  # Deno (JSR)
```

## Documentation

See the [Panqueue repository](https://github.com/50BytesOfJohn/panqueue) for full documentation.
