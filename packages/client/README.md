# @panqueue/client

Type-safe producer client for enqueuing [Panqueue](https://github.com/50BytesOfJohn/panqueue) jobs into Redis — typed payloads, at-least-once delivery, atomic Lua scripts.

## Install

```sh
npm install @panqueue/client   # or pnpm / yarn / bun
deno add jsr:@panqueue/client  # Deno (JSR)
```

## Documentation

See the [Panqueue repository](https://github.com/50BytesOfJohn/panqueue) for full documentation.
