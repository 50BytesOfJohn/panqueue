# @panqueue/config

Type-safe shared configuration for [Panqueue](https://github.com/50BytesOfJohn/panqueue) clients and workers — define your queues and payload types once, and `enqueue`/`defineWorker` infer the right shapes everywhere.

## Install

```sh
npm install @panqueue/config   # or pnpm / yarn / bun
deno add jsr:@panqueue/config  # Deno (JSR)
```

## Documentation

See the [Panqueue repository](https://github.com/50BytesOfJohn/panqueue) for full documentation.
