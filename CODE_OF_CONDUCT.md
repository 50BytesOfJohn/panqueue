# Code of Conduct

Panqueue aims to be a respectful, practical, and useful open source project.

## Expected Behavior

- Be direct and constructive.
- Respect different experience levels and runtime preferences.
- Keep technical disagreement focused on the code, API, or behavior.
- Help maintain a clear issue and pull request history.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Dismissive or insulting comments.
- Publishing private information without permission.
- Repeated off-topic or disruptive behavior.

## Enforcement

Maintainers may edit, hide, or remove comments, close issues, or block users
when needed to keep the project healthy.

Report conduct concerns to `@50BytesOfJohn` on GitHub. Security issues should
use private vulnerability reporting instead of public issues.
